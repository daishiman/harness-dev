## 4. 実行仕様

### 4.1 思考プロセス

| ステップ | アクション                                     |
| -------- | ---------------------------------------------- |
| 1        | AI利用の有無・範囲を確認                       |
| 2        | 使用するAIモデル・サービスを特定               |
| 3        | AI生成物の著作権帰属を整理                     |
| 4        | プロンプトの著作権帰属を検討                   |
| 5        | ハルシネーション等のリスク免責条項を設計       |
| 6        | データ管理条項（入力データ・学習データ）を設計 |
| 7        | 個人情報保護・機密情報管理条項を設計           |
| 8        | 最新のAI関連ガイドラインをウェブ検索で確認 [WebSearch] |

### 4.2 AI生成物の著作権判断基準

| 創作的寄与                       | 著作権の帰属                       |
| -------------------------------- | ---------------------------------- |
| AIが完全に自動生成               | 著作権なし（著作物性なし）         |
| 人間がAIを道具として使用、創作的意図・寄与あり | 著作権あり得る（使用者に帰属）     |
| プロンプトが詳細かつ具体的       | 著作権あり得る（プロンプト作成者に帰属） |
| 生成物を選択・修正・編集         | 著作権あり得る（編集者に帰属）     |

### 4.3 免責条項テンプレート

```markdown
### AI利用に関する特則

1. 甲は、以下の事項を了承した上で、本業務を委託する。
   （1）生成AIの出力結果は、使用するモデル、バージョン、パラメータ設定等により異なる場合があること
   （2）同一のプロンプトを使用しても、異なる出力が得られる場合があること
   （3）AI出力には、事実と異なる内容（ハルシネーション）が含まれる可能性があること
   （4）AI出力の一部又は全部が、第三者の著作物と類似する可能性を完全に排除できないこと
   （5）AI出力の品質・精度には一定の限界があること

2. 乙は、前項の特性に起因する損害について、乙に故意又は重大な過失がある場合を除き、責任を負わない。

3. 甲は、AI出力を含む成果物の利用にあたっては、自己の責任において内容を確認し、使用するものとする。
```
