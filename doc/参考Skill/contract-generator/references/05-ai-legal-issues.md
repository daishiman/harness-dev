# 第5章: AI特有の法的問題と対応

## 5.1 AI生成物の著作権

### 文化庁の見解（「AIと著作権に関する考え方について」令和6年3月）

#### AI生成物に著作権が発生するか
- AIが完全に自動生成：著作権なし（人間の創作的寄与がないため）
- 人間が道具としてAIを使用し、創作的意図・創作的寄与あり：著作権あり得る

#### 創作的寄与の判断要素
- プロンプトの詳細さと具体性
- 生成物の選択・修正・編集の程度
- 創作過程への人間の関与度

#### 著作権法第30条の4（AI学習における権利制限）
- 著作物の思想・感情の享受を目的としない利用は許諾不要
- ただし、著作権者の利益を不当に害する場合は適用除外

## 5.2 AI出力の品質保証問題

### AI出力の特性
- 同一プロンプトでも異なる出力が得られる可能性
- ハルシネーション（事実と異なる情報の生成）のリスク
- 第三者の著作物に類似する可能性

### 契約上の対応
```
【条項例：AI利用に関する免責】
第○条（AI利用に関する特則）
1 甲は、以下の事項を了承した上で、本業務を委託する。
  （1）生成AIの出力結果は、使用するモデル、バージョン、
      パラメータ設定等により異なる場合があること
  （2）同一のプロンプトを使用しても、異なる出力が得られる場合があること
  （3）AI出力には、事実と異なる内容（ハルシネーション）が
      含まれる可能性があること
  （4）AI出力の一部又は全部が、第三者の著作物と類似する
      可能性を完全に排除できないこと
  （5）AI出力の品質・精度には一定の限界があること

2 乙は、前項の特性に起因する損害について、
  乙に故意又は重大な過失がある場合を除き、責任を負わない。

3 甲は、AI出力を含む成果物の利用にあたっては、
  自己の責任において内容を確認し、使用するものとする。
```

## 5.3 AI著作権侵害リスクへの対応

### 侵害が成立する要件
- 類似性：既存著作物の「表現上の本質的特徴」を直接感得できること
- 依拠性：既存著作物に基づいて作成されたこと

### 契約上の対応
```
【条項例：第三者権利侵害への対応】
第○条（第三者の権利侵害）
1 第三者から甲に対し、成果物が当該第三者の著作権その他の
  権利を侵害する旨の請求がなされた場合、甲は直ちに乙に
  書面により通知する。
2 前項の場合、甲乙協議の上、対応方針を決定する。
3 成果物が第三者の権利を侵害していることが判明した場合、
  乙は、甲の選択に従い、次の各号のいずれかの措置を講じる。
  （1）権利侵害部分の修正又は代替物の提供
  （2）当該第三者から利用許諾を取得
  （3）報酬の返還
4 前項の措置に要する費用は、乙に故意又は重大な過失がある
  場合を除き、甲乙折半で負担する。
```

## 5.4 データ管理条項

### 入力データの取扱い
```
【条項例】
第○条（データの取扱い）
1 甲が乙に提供するデータ（以下「甲データ」という。）の
  知的財産権その他一切の権利は、甲に帰属する。
2 乙は、甲データを本業務の目的にのみ使用し、
  甲の書面による事前の承諾なく第三者に開示してはならない。
3 乙は、甲データをAIモデルの学習データとして使用しない。
4 乙は、本契約終了後速やかに、甲データを返還又は廃棄する。
```

### 個人情報の取扱い
```
【条項例】
第○条（個人情報の保護）
1 甲は、本業務のために乙に個人情報を提供する場合、
  事前に本人の同意を得るものとする。
2 乙は、甲から提供された個人情報を、本業務の目的にのみ使用し、
  個人情報の保護に関する法律その他関連法令を遵守して管理する。
3 甲は、AIに入力するデータから、可能な限り個人情報を
  除去又は匿名化するものとする。
```
