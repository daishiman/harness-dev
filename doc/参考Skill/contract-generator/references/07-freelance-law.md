# 第7章: フリーランス新法（2024年）対応

## 7.1 適用対象

### 特定受託事業者（フリーランス）の定義
- 従業員を使用しない個人事業主
- 一人法人（代表者のみ）

### 従業員の定義
- 週20時間以上かつ31日以上雇用見込みの労働者

## 7.2 発注事業者の義務

### 【全ての発注事業者】

| 義務 | 内容 | 条文 |
|-----|------|-----|
| 取引条件の明示 | 業務内容、報酬額、支払期日等を書面等で明示 | 3条 |
| 支払期日の設定 | 成果物受領日から**60日以内** | 4条 |

### 【特定業務委託事業者（1か月以上の継続取引）】

| 禁止行為 | 内容 | 条文 |
|---------|------|-----|
| 受領拒否 | 正当な理由なく受領を拒否 | 5条 |
| 報酬減額 | 正当な理由なく報酬を減額 | 5条 |
| 返品 | 正当な理由なく返品 | 5条 |
| 買いたたき | 著しく低い報酬額の設定 | 5条 |
| 購入・利用強制 | 不要な物品等の購入・利用を強制 | 5条 |
| 不当な経済上の利益提供要請 | 金銭等の提供を要請 | 5条 |
| やり直し強制 | 正当な理由なくやり直しを強制 | 5条 |

### 【6か月以上の継続取引】

| 義務 | 内容 | 条文 |
|-----|------|-----|
| 中途解除・不更新の予告 | **30日前**までに予告 | 11条 |
| 理由開示 | 予告日から解除日までに請求あれば開示 | 11条 |
| 育児介護両立配慮 | 申出に応じた必要な配慮（努力義務） | 13条 |

## 7.3 法第3条 明示事項13項目【必須】

フリーランス新法第3条および公正取引委員会規則に基づき、発注者は以下の**13項目**を書面等で明示する義務がある。

### 13項目一覧

| # | 項目 | 条文根拠 | 記載例 |
|---|------|----------|--------|
| 1 | **発注事業者の氏名又は名称** | 法3条1項1号 | 株式会社○○ |
| 2 | **特定受託事業者の氏名又は名称** | 規則1条1号 | ○○太郎 |
| 3 | **業務委託をした日** | 法3条1項2号 | 令和○年○月○日 |
| 4 | **給付の内容（業務内容）** | 法3条1項3号 | AIコンサルティング、研修等 |
| 5 | **給付を受領する期日（納期）** | 法3条1項4号 | 個別契約にて定める |
| 6 | **給付を受領する場所（納品場所）** | 法3条1項5号 | 電子メール又は甲指定の方法 |
| 7 | **検査を完了する期日** | 法3条1項6号 | 成果物受領後7日以内 |
| 8 | **報酬の額** | 法3条1項7号 | 個別契約にて定める |
| 9 | **報酬の支払期日** | 法3条1項7号 | 翌月末日（60日以内） |
| 10 | **現金以外の支払方法** | 規則1条2号 | 該当なし（現金払い）※手形等の場合は金額・満期を記載 |
| 11 | **一括決済方式の利用の有無** | 規則1条3号 | 利用しない |
| 12 | **報酬の支払方法** | 規則1条4号 | 銀行振込（手数料は甲負担） |
| 13 | **知的財産権の帰属** | 法3条2項 | 報酬完済で甲に移転（著作権条項参照） |

### 注意事項

- **項目10〜11**: 該当しない場合でも「該当なし」「利用しない」と明記する
- **項目13**: 著作権譲渡がある場合は必須。譲渡対価が報酬に含まれることも明記
- 13項目は**別紙「取引条件明示書」**として契約書に添付することを推奨

### 【別紙】取引条件明示書テンプレート

```markdown
## 【別紙】取引条件明示書

*特定受託事業者に係る取引の適正化等に関する法律第3条に基づく取引条件の明示*

**1. 発注事業者の氏名又は名称**　○○株式会社

**2. 特定受託事業者の氏名又は名称**　○○○○

**3. 業務委託をした日**　令和　　年　　月　　日

**4. 給付の内容（業務内容）**　○○業務（詳細は個別契約にて定める）

**5. 給付を受領する期日（納期）**　個別契約にて定める

**6. 給付を受領する場所（納品場所）**　電子メール又は甲が別途指定する方法による

**7. 検査を完了する期日**　成果物受領後○日以内（第○条第○項に基づく）

**8. 報酬の額**　個別契約にて定める

**9. 報酬の支払期日**　請求書受領月の翌月末日（役務提供又は成果物受領から60日以内）

**10. 現金以外の支払方法**　該当なし（現金払い）

**11. 一括決済方式の利用の有無**　利用しない

**12. 報酬の支払方法**　銀行振込（振込手数料は甲負担）

**13. 知的財産権の帰属**　成果物の著作権は報酬完済をもって甲に移転（第○条に基づく）
```

## 7.4 契約書での対応

### 取引条件の明示
```
【条項例】
第○条（取引条件の明示）
甲は、個別契約を締結するにあたり、乙に対し、特定受託事業者に係る
取引の適正化等に関する法律第3条に定める事項を書面又は電磁的方法
により明示する。明示事項の詳細は【別紙】取引条件明示書に定める。
```

### 支払期日
```
【条項例】
第○条（支払期日）
甲は、成果物を受領した日から60日以内のできる限り短い期間内に、
乙に対し、報酬を支払う。
```

### 中途解除の予告（6か月以上の契約）
```
【条項例】
第○条（中途解除の予告）
1 甲が本契約を中途解除し、又は更新しないこととする場合、
  甲は乙に対し、少なくとも30日前までに、その旨を予告する。
2 乙は、前項の予告を受けた場合、甲に対し、解除又は不更新の
  理由の開示を求めることができ、甲はこれに応じるものとする。
```

## 7.4 フリーランス新法チェックリスト

- [ ] 支払期日は成果物受領から**60日以内**か
- [ ] 取引条件（業務内容、報酬額、支払期日等）を**書面等で明示**しているか
- [ ] 禁止行為（受領拒否、報酬減額、返品等）を回避しているか
- [ ] 6か月以上の契約の場合、中途解除・不更新の予告規定があるか
- [ ] 解除の**30日前予告**の規定があるか
- [ ] 理由開示請求への対応規定があるか
