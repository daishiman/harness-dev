# 第10章: チェックリストと実務フロー

## 10.1 契約書作成チェックリスト

### 【契約全体】
- [ ] 契約当事者の正式名称・住所は正確か
- [ ] 契約の目的・適用範囲は明確か
- [ ] 契約類型（請負/準委任）は業務内容に適合しているか
- [ ] 基本契約と個別契約の優先関係は定められているか
- [ ] 個別契約の成立要件は明確か

### 【業務内容】
- [ ] 業務の範囲・内容は具体的か
- [ ] 成果物の定義は明確か
- [ ] 納入方法・検収基準は定められているか
- [ ] AI利用に関する説明・免責は記載されているか
- [ ] 再委託の可否・条件は定められているか

### 【報酬・支払】
- [ ] 報酬額の算定方法は明確か
- [ ] 支払条件（締日・支払日・方法）は定められているか
- [ ] フリーランス新法の支払期日規制（**60日以内**）に適合しているか
- [ ] 遅延損害金の規定はあるか

### 【知的財産権】⭐最重要
- [ ] 著作権の帰属は明確か
- [ ] **「著作権法第27条及び第28条の権利を含む」は明記されているか**
- [ ] 著作者人格権不行使の規定はあるか
- [ ] 既存の知的財産権の扱いは定められているか
- [ ] ノウハウ・プロンプトの取扱いは定められているか

### 【責任】
- [ ] 契約不適合責任（請負）/ 善管注意義務（準委任）の内容・期間は定められているか
- [ ] 損害賠償の範囲・上限は定められているか
- [ ] 免責事項は明確か
- [ ] AI特有のリスク（ハルシネーション等）への対応はあるか

### 【秘密保持】
- [ ] 秘密情報の定義は明確か
- [ ] 例外（公知情報等）は定められているか
- [ ] 秘密保持義務の存続期間は定められているか

### 【契約終了】
- [ ] 契約期間・更新条件は定められているか
- [ ] 中途解約の条件は定められているか
- [ ] 解除事由（催告解除・無催告解除）は明確か
- [ ] 契約終了後の処理は定められているか
- [ ] 存続条項は明記されているか

### 【その他】⭐必須
- [ ] **反社会的勢力排除条項はあるか**
- [ ] 権利義務の譲渡禁止条項はあるか
- [ ] 合意管轄条項はあるか
- [ ] 協議条項はあるか

## 10.2 契約書作成の実務フロー

```
1. 取引類型の特定
   ↓
   ・売買か、請負か、準委任か
   ・AI事業の場合、多くは準委任型

2. 基本条項の作成
   ↓
   ・目的、定義、個別契約の成立

3. 業務関連条項の作成
   ↓
   ・業務内容、納入、検収、報酬

4. 知的財産権条項の作成【最重要】
   ↓
   ・著作権帰属（27条・28条明記）
   ・著作者人格権不行使
   ・AI利用特則

5. 責任関連条項の作成
   ↓
   ・契約不適合/善管注意義務
   ・損害賠償（範囲・上限）
   ・免責

6. 秘密保持条項の作成
   ↓
   ・秘密情報の定義・除外
   ・保持義務・存続期間

7. 契約終了関連条項の作成
   ↓
   ・期間、解約、解除
   ・終了後処理、存続条項

8. 一般条項の追加
   ↓
   ・反社排除【必須】
   ・譲渡禁止、協議、管轄

9. フリーランス新法対応確認
   ↓
   ・支払期日（60日以内）
   ・解約予告（30日前）※6か月以上の場合

10. 最終チェック・レビュー
```

## 10.3 税務チェックリスト

| 項目 | チェック内容 | 判定 |
|------|-------------|------|
| 印紙税 | 契約期間3か月超 or 自動更新あり → 4,000円 | □ |
| 印紙税 | 電子契約の場合 → 不要 | □ |
| 源泉徴収 | デザイン・著述・講演 → 対象 | □ |
| 源泉徴収 | コンサル・システム開発 → 非対象 | □ |
| インボイス | 登録済み → 登録番号記載 | □ |
| インボイス | 未登録 → 経過措置説明 | □ |
