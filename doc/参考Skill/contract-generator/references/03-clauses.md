## 3.2 主要条項の詳細と条項例

### 3.2.1 目的条項
```
【条項例】
第1条（目的）
本契約は、甲が乙に対して委託するAIコンサルティング、プロンプト提供、
AI研修等の業務（以下「本業務」という。）に関する基本的な事項を定め、
甲乙間の継続的な取引の円滑な実施を図ることを目的とする。
```

### 3.2.2 定義条項
```
【条項例】
第2条（定義）
本契約において、次の各号に掲げる用語の意義は、当該各号に定めるところによる。
（1）「本業務」：甲が乙に委託するAIコンサルティング、プロンプト提供、
    AI研修その他これらに関連する業務をいう。
（2）「成果物」：本業務の遂行により作成されるレポート、プロンプト、
    研修資料、分析結果その他一切の制作物をいう。
（3）「プロンプト」：生成AIに対して指示・質問を行うための入力文をいう。
（4）「AI出力」：生成AIが出力したテキスト、画像、コード等をいう。
（5）「ノウハウ」：本業務に関連して乙が保有するアイデア、手法、
    技術情報、経験則等をいう。
（6）「生成AI」：OpenAI社のGPT、Anthropic社のClaude、Google社のGemini
    その他のテキスト・画像等を生成する人工知能システムをいう。
```

### 3.2.3 再委託条項
```
【条項例】
第○条（再委託）
1 乙は、甲の書面による事前の承諾なく、本業務の全部又は一部を
  第三者に再委託してはならない。
2 前項の承諾を得て再委託する場合、乙は、再委託先に対し、
  本契約における乙の義務と同等の義務を負わせるものとし、
  再委託先の行為について甲に対し一切の責任を負う。
3 乙は、再委託先の名称、再委託する業務の範囲その他甲が
  求める事項を甲に報告する。
```

### 3.2.4 検査・検収条項（民法改正対応版）
```
【条項例】
第○条（検査・検収）
1 甲は、成果物の納入を受けたときは、納入後10営業日以内に、
  成果物が個別契約の内容に適合するか検査を行う。
2 甲は、前項の検査の結果、成果物が個別契約の内容に適合しない
  （以下「契約不適合」という。）と認めた場合、乙に対し、
  当該検査後5営業日以内に、書面により通知する。
3 甲が第1項の期限内に検査を行わなかった場合、又は第2項の期限内に
  通知を行わなかった場合は、検査に合格したものとみなし、検収完了とする。
4 乙は、第2項の通知を受けた場合、速やかに成果物の修補、
  代替物の引渡し又は不足分の引渡しを行うものとする。
```

### 3.2.5 報酬・支払条項
```
【条項例】
第○条（報酬及び支払方法）
1 甲は、個別契約に定める報酬額を、乙に支払う。
2 報酬の支払は、検収完了月の末日締め、翌月末日までに、
  乙が指定する金融機関の口座に振り込む方法により行う。
  振込手数料は甲の負担とする。
3 甲及び乙は、協議の上、報酬額を変更することができる。
4 本業務の遂行に要する交通費、宿泊費その他の実費は、
  事前に甲の承諾を得た場合に限り、甲が別途負担する。
```

### 3.2.6 契約不適合責任条項（乙を守る構成）
```
【条項例】
第○条（契約不適合責任）
1 甲は、成果物又は役務が本契約及び個別契約の内容に適合せず、
  かつ本業務の目的達成を妨げる場合に限り、検収完了後30日以内に
  乙に通知することにより、乙に対し、追完（修正、補修、代替物の
  納品等）を請求することができる。
2 乙は、前項の追完請求を受けた場合、合理的な範囲で追完を行う。
  追完の方法は乙が選択するものとする。
3 前2項の場合において、甲が相当の期間を定めて履行の追完の催告をし、
  その期間内に履行の追完がないときは、甲は、その不適合の程度に応じて
  報酬の減額を請求することができる。
4 前3項の規定は、甲が乙に対し、損害賠償の請求及び契約の解除を
  することを妨げない。
5 甲は、契約不適合を知った時から1年以内にその旨を乙に通知しないときは、
  その契約不適合を理由として、履行の追完の請求、報酬の減額の請求、
  損害賠償の請求及び契約の解除をすることができない。
```

**設計ポイント:**
- 「軽微な不適合」という曖昧な表現を避け、「目的達成を妨げる場合に限り」で乙を守る
- 追完方法は乙が選択（甲の指示通りにやる義務なし）
- 「合理的な範囲で」追完（過度な対応要求を防ぐ）

### 3.2.7 損害賠償条項（ファクトベース構成）
```
【条項例】
第○条（損害賠償）
1 甲又は乙は、本契約又は個別契約に違反し、相手方に損害を与えた場合、
  相手方に対し、現実に生じた通常の損害を賠償する責任を負う。

2 前項の損害賠償の範囲は通常損害に限り、特別の事情によって生じた損害
  （特別損害）、逸失利益及び間接損害は含まないものとする。

3 本契約又は個別契約に基づく乙の損害賠償責任の総額は、
  当該損害の原因となった個別契約に定める報酬額を上限とする。

4 前2項の規定は、故意又は重大な過失がある場合には適用しない。
```

**設計ポイント:**
- 「現実に生じた」：実際に発生した損害の立証が必要（仮定的・将来の損害を除外）
- 「通常の損害」：民法416条1項の通常損害に限定（特別損害を明示的に除外）
- 第2項で除外対象を明確化：特別損害・逸失利益・間接損害を列挙
- ファクトベースで争いやすい構成：損害の存在と因果関係を具体的に立証する必要あり

### 3.2.8 解除条項（民法改正対応版）
```
【条項例】
第○条（解除）
1 甲又は乙は、相手方が本契約又は個別契約に違反し、
  相当期間を定めて是正を催告したにもかかわらず、
  当該期間内に是正されない場合、本契約及び個別契約の
  全部又は一部を解除することができる。
  ただし、当該違反が軽微であるときは、この限りでない。

2 甲又は乙は、相手方が次の各号のいずれかに該当した場合、
  何らの催告なく、直ちに本契約及び個別契約の全部又は一部を
  解除することができる。
  （1）支払停止若しくは支払不能に陥ったとき、又は手形若しくは
      小切手が不渡りになったとき
  （2）破産手続開始、民事再生手続開始、会社更生手続開始又は
      特別清算開始の申立てがあったとき
  （3）差押え、仮差押え、仮処分、強制執行又は競売の申立てを受けたとき
  （4）公租公課の滞納処分を受けたとき
  （5）監督官庁より営業許可の取消し、停止その他の行政処分を受けたとき
  （6）解散、事業の全部又は重要な一部の譲渡を決議したとき
  （7）資産又は信用状態に重大な変化が生じ、本契約の履行が
      困難になるおそれがあると認められるとき
  （8）本契約に定める重大な義務に違反したとき

3 前2項による解除は、損害賠償の請求を妨げない。
```
