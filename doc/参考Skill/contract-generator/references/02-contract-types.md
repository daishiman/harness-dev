# 第2章: 契約類型の選択（請負vs準委任）

## 2.1 契約類型の重要性

AI事業において最も重要な判断の一つが、**請負契約**と**準委任契約**のどちらを選択するかである。契約類型により、責任の範囲、報酬請求権、解除権などが大きく異なる。

## 2.2 請負契約と準委任契約の比較

| 項目 | 請負契約 | 準委任契約 |
|-----|---------|----------|
| **民法規定** | 民法632条 | 民法656条（643条準用） |
| **目的** | 仕事の完成 | 事務処理の遂行 |
| **完成義務** | あり | なし |
| **報酬発生** | 仕事完成時 | 履行割合型：履行の割合に応じて / 成果完成型：成果達成時 |
| **責任** | 契約不適合責任 | 善管注意義務 |
| **再委託** | 原則可能 | 原則禁止（民法644条の2） |
| **中途解約** | 注文者はいつでも解除可（損害賠償必要） | 両者いつでも解除可（不利な時期は損害賠償） |

## 2.3 AI事業における契約類型の選択基準

| 業務内容 | 推奨契約類型 | 理由 |
|---------|------------|------|
| AIコンサルティング | **準委任契約** | 成果の達成を約束できない |
| AI研修・セミナー | **準委任契約** | 講師としての役務提供が主 |
| プロンプト提供（カスタム） | **請負契約** | 特定の成果物を納品 |
| プロンプトテンプレート販売 | **売買契約** | 既製品の販売 |
| AI導入支援 | **準委任契約** | 支援行為の提供が主 |
| AIシステム開発 | **請負契約** | システムの完成が目的 |

## 2.4 準委任契約の2つの類型（2020年民法改正）

### ①履行割合型（民法648条2項）
- 委任事務の履行の割合に応じて報酬を請求できる
- 時間単価や月額固定報酬の形態に適合
- 成果が出なくても、業務遂行すれば報酬発生

### ②成果完成型（民法648条の2）
- 委任事務の履行により得られる成果に対して報酬を支払う
- 成功報酬型の契約に適合
- **注意:** 仕事の完成責任は負わない（請負との違い）

```
【条項例：成果完成型】
第○条（報酬）
1 本業務の報酬は、本業務の履行により得られる成果に対して
  支払われるものとし、その金額は個別契約において定める。
2 前項の成果が達成されない場合、乙は報酬を請求できない。
  ただし、甲の責めに帰すべき事由により成果が達成されない
  場合は、この限りでない。
```

## 2.5 善管注意義務（準委任契約の核心）

**定義（民法644条）:**
> 受任者は、委任の本旨に従い、善良な管理者の注意をもって、委任事務を処理する義務を負う。

### 善管注意義務の内容
- 行為者の階層、地位、職業に応じて、社会通念上、客観的・一般的に要求される注意を払う義務
- **結果責任ではなく過程の責任**（結果が出なくても善管注意義務を果たしていれば責任なし）

### AI事業における具体例
- AIコンサルタント：業界標準の知識・技術に基づく助言を提供する義務
- AI研修講師：適切な内容の研修を誠実に実施する義務
- プロンプトエンジニア：適切な技術・知見に基づきプロンプトを作成する義務

```
【条項例：善管注意義務】
第○条（善管注意義務）
乙は、本業務の本旨に従い、善良な管理者の注意をもって
本業務を遂行する。
```

### 善管注意義務違反の効果
- 債務不履行に基づく損害賠償責任
- 契約解除の対象

## 2.6 契約不適合責任（請負契約の核心）

### 2020年民法改正による変更

| 項目 | 旧民法（瑕疵担保責任） | 改正民法（契約不適合責任） |
|-----|---------------------|------------------------|
| 要件 | 「隠れた瑕疵」 | 「契約の内容に適合しない」 |
| 権利 | 損害賠償・解除のみ | ①追完請求 ②代金減額 ③損害賠償 ④解除 |
| 損害範囲 | 信頼利益 | 履行利益まで拡大 |
| 期間 | 知った時から1年以内に行使 | 知った時から1年以内に**通知** |
