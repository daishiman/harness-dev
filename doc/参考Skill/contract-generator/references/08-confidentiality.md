# 第8章: 秘密保持条項の詳細

## 8.1 秘密情報の定義

### 秘密情報に含めるべき事項
- 技術情報（設計、仕様、プログラム、データ、ノウハウ等）
- 営業情報（顧客情報、取引先情報、価格情報、事業計画等）
- 財務情報
- 人事情報
- 書面等により「秘密」と明示された情報
- 口頭で開示され、秘密である旨告知された情報

### 除外事項
- 開示を受けた時点で既に公知であった情報
- 開示を受けた時点で受領者が既に保有していた情報
- 開示を受けた後、受領者の責めに帰すべき事由によらず公知となった情報
- 受領者が正当な権限を有する第三者から秘密保持義務を負うことなく適法に取得した情報
- 受領者が開示を受けた情報に依拠せず独自に開発した情報

## 8.2 秘密保持条項の条項例

```
【条項例】
第○条（秘密保持）
1 本契約において「秘密情報」とは、本契約に関連して、
  一方当事者（以下「開示者」という。）が他方当事者
  （以下「受領者」という。）に開示する次の各号に掲げる情報をいう。
  （1）技術情報（設計、仕様、プログラム、データ、ノウハウ等）
  （2）営業情報（顧客情報、取引先情報、価格情報、事業計画等）
  （3）財務情報
  （4）人事情報
  （5）書面等により「秘密」又は「Confidential」と明示された情報
  （6）その他口頭又は視覚的手段により開示され、開示時に秘密である旨
      告知された情報

2 前項の規定にかかわらず、次の各号のいずれかに該当する情報は、
  秘密情報に含まれない。
  （1）開示を受けた時点で既に公知であった情報
  （2）開示を受けた時点で受領者が既に保有していた情報
  （3）開示を受けた後、受領者の責めに帰すべき事由によらず公知となった情報
  （4）受領者が正当な権限を有する第三者から秘密保持義務を負うことなく
      適法に取得した情報
  （5）受領者が開示を受けた情報に依拠せず独自に開発した情報

3 受領者は、秘密情報を善良なる管理者の注意をもって管理し、
  開示者の書面による事前の承諾なく、第三者に開示又は漏洩してはならない。

4 受領者は、秘密情報を本業務の目的以外に使用してはならない。

5 前2項の規定にかかわらず、受領者は、次の各号に該当する場合、
  必要最小限の範囲で秘密情報を開示することができる。
  （1）法令又は裁判所若しくは行政機関の命令により開示が義務付けられた場合
  （2）自己の役員及び従業員で、秘密情報を知る必要のある者に開示する場合
  （3）本契約に基づく再委託先に開示する場合

6 本条の義務は、本契約終了後3年間存続する。
```

## 8.3 NDA（秘密保持契約）の印紙税

**原則:** 秘密保持契約単体は課税文書に該当しない → **印紙税不要**

**例外:** 請負や継続的取引を含む混合契約の場合は課税対象となりうる

## 8.4 秘密保持条項のバリエーション

### 開示範囲の制限
```
受領者は、秘密情報を、本業務の遂行のために必要な範囲で、
自己の役員及び従業員（以下「役職員」という。）に限り開示することができる。
この場合、受領者は、当該役職員に対し、本条に定める秘密保持義務と
同等の義務を課すものとする。
```

### 返還・廃棄義務
```
受領者は、開示者の書面による請求があった場合、又は本契約が終了した場合、
速やかに秘密情報（その複製物を含む。）を開示者に返還し、
又は開示者の指示に従い廃棄する。
```

### 存続期間のバリエーション
| 期間 | 適用場面 |
|------|---------|
| 1年 | 一般的なビジネス情報 |
| 3年 | 技術情報・営業秘密 |
| 5年以上 | 重要な営業秘密 |
| 無期限 | 個人情報・特に機密性の高い情報 |
