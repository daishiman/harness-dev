# 第1章: 取引基本契約書の基礎

## 1.1 取引基本契約書とは

取引基本契約書とは、同じ相手と反復継続的に取引を行う場合に、個々の取引に共通して適用される基本的な契約条件をあらかじめ定めておく契約書。

### 別名
- 売買基本契約書
- 業務委託基本契約書
- 製造委託基本契約書
- 継続的取引基本契約書
- 外注基本契約書
- 開発基本契約書

### 目的
- 取引条件の明確化によるトラブル防止
- 個別交渉の省力化・取引の迅速化
- リスクの予測と回避方法の事前確保
- 債権回収の確実性向上

### 構造
```
取引基本契約書（共通ルール）
    ↓ 適用
個別契約（発注書・注文請書）
```

## 1.2 基本契約と個別契約の関係

**優先順位の明確化が必須**

```
【条項例】
第○条（基本契約と個別契約の関係）
1 甲及び乙は、個別契約において、本基本契約と異なる事項を
  定めることができる。
2 本基本契約と個別契約の内容が異なる場合は、個別契約の定めが
  本基本契約に優先する。
```

### 個別契約優先が一般的な理由
- 個別の取引で特殊な条件が必要な場合に対応可能
- 基本契約締結後の状況変化への柔軟な対応
- 実務上のニーズに即した運用が可能

## 1.3 個別契約の成立要件

```
【条項例】
第○条（個別契約の成立）
1 個別契約は、甲が乙に対し、発注日、業務内容、報酬額、
  納入期日、納入場所等を記載した発注書を交付し、
  乙がこれを承諾することにより成立する。
2 乙は、発注書の受領後5営業日以内に甲に対して書面により
  異議を述べない場合、当該発注を承諾したものとみなす。
3 甲及び乙は、前項の期間経過前であっても、乙が発注書記載の
  業務に着手した場合は、個別契約が成立したものとみなす。
```

## 1.4 印紙税

### 取引基本契約書の印紙税

| 契約期間 | 印紙税額 |
|---------|---------|
| 3か月超または更新条項あり | 4,000円/通（第7号文書） |
| 3か月以内で更新なし | 非課税 |

**電子契約の場合:** 印紙税不要（課税文書に該当しない）
