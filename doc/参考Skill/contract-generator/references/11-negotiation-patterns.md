# 法務課との交渉パターン

> このドキュメントは、取引先の法務課からの修正依頼に対する交渉パターンを整理したものです。
> 乙（フリーランス・受注者）の立場を守りながら、相手の要望を受け入れる方法を示します。

---

## 概要

| カテゴリ | パターン数 | 目的 |
|---------|-----------|------|
| AI責任条項 | 3 | AI利用時のリスク分担を明確化 |
| 著作権条項 | 2 | 著作権譲渡と実績紹介の両立 |
| 損害賠償条項 | 3 | 責任範囲の限定 |
| 契約期間条項 | 2 | 継続性と柔軟性の確保 |
| 解除条項 | 3 | 双方の公平性担保 |

---

## 1. AI責任条項の交渉パターン

### パターン1-1: 過失範囲の交渉

**法務課の要望:**
「重大な過失」だけでなく「過失」も含めてほしい

**乙の対案:**
```
・「直接的な故意又は過失」に変更
・損害賠償上限条項への参照を追加
```

**条文例:**
> ただし、前項各号に定めるリスクに起因して甲又は第三者に生じた損害については、乙の**直接的な**故意又は過失による場合を除き、乙は責任を負わないものとする。**なお、本項に基づく乙の損害賠償責任は、第○条の規定に従うものとする。**

**交渉ポイント:**
- 「直接的な」を追加することで、乙が直接関与していない損害（運用ミス等）を除外
- 損害賠償上限条項への参照で、万が一の責任も報酬額が上限

---

### パターン1-2: 成果物帰責性の限定

**法務課の要望:**
「乙に帰責性がある場合はこの限りではない」を追記

**乙の対案:**
```
・「成果物に関して乙に帰責性がある場合」に限定
```

**条文例:**
> 確認を怠ったことにより甲又は第三者に生じた損害について、乙は責任を負わないものとする。**ただし、成果物に関して**乙に帰責性がある場合はこの限りではない。

**交渉ポイント:**
- 「成果物に関して」を追加し、納品物に起因する問題に限定
- 研修中の口頭説明等、成果物以外の業務は対象外

---

### パターン1-3: AI費用負担の明確化

**法務課の要望:**
AI有料アカウント費用の負担を明記

**乙の対案:**
```
・各自が自己の使用分を負担
```

**条文例:**
> 本業務の遂行に必要な生成AIツール（ChatGPT、Claude、Gemini等）の有料アカウント費用は、**甲及び乙がそれぞれ自己の使用分を負担する**ものとする。

**交渉ポイント:**
- 甲（発注者）の社内利用分は甲負担
- 乙（講師）の準備・実施分は乙負担
- 双方にとって公平な分担

---

## 2. 著作権条項の交渉パターン

### パターン2-1: 著作者人格権不行使 + ポートフォリオ例外

**法務課の要望:**
著作者人格権（公表権・氏名表示権・同一性保持権）を行使しない条項を追加

**乙の対案:**
```
・不行使条項を受け入れつつ
・ポートフォリオ・実績紹介の例外を追加
```

**条文例:**
> 乙は、前項の規定に基づき甲に著作権その他の知的財産権を譲渡した成果物につき、著作者人格権（公表権、氏名表示権、同一性保持権）その他の人格権を行使しないものとする。**ただし、乙が自己のポートフォリオ又は実績紹介の目的で成果物の概要を公表することは、甲の秘密情報を含まない範囲において妨げられないものとする。**

**交渉ポイント:**
- 業界標準の不行使条項は受け入れ
- 営業活動に必要な実績紹介は確保
- 「秘密情報を含まない範囲」で甲の利益も保護

---

### パターン2-2: 著作権譲渡対価の明記

**法務課の要望:**
著作権譲渡対価が報酬に含まれることを明記（フリーランス新法対応）

**乙の対案:**
```
・そのまま受け入れ（フリーランス保護法として有利）
```

**条文例:**
> 本業務により生じた成果物に関する著作権（著作権法第27条及び第28条の権利を含む。）の創作は、第2条に定める本業務に含まれる。**当該著作権の譲渡対価は、第2項の場合を除いて第○条に定める報酬に含まれる。**

**交渉ポイント:**
- フリーランス新法で明示が義務化されている事項
- 乙にとって不利ではなく、むしろ明確化として有利

---

## 3. 損害賠償条項の交渉パターン

### パターン3-1: 契約不適合責任の通知期間

**法務課の要望:**
通知期間を30日から60日に延長

**乙の対案:**
```
・60日を受け入れ（民法原則よりは短い）
```

**条文例:**
> 甲は、成果物又は役務が本契約及び個別契約の内容に適合せず、かつ本業務の目的達成を妨げる場合に限り、検収完了後**60日以内**に乙に通知することにより、乙に対し、追完を請求することができる。

**交渉ポイント:**
- 民法上の原則は「知った時から1年以内」
- 60日でも乙に有利な設定
- 「目的達成を妨げる場合に限り」の条件は維持

---

### パターン3-2: 損害賠償上限の設定

**法務課の要望:**
損害賠償の範囲・上限を明確化

**乙の対案:**
```
・通常損害に限定
・上限を報酬額に設定
・重過失の場合は上限適用除外
```

**条文例:**
> 1. 甲又は乙は、本契約又は個別契約に違反し、相手方に損害を与えた場合、相手方に対し、**現実に生じた通常の損害**を賠償する責任を負う。
> 2. 前項の損害賠償の範囲は通常損害に限り、**特別の事情によって生じた損害（特別損害）、逸失利益及び間接損害は含まない**ものとする。
> 3. 本契約及び個別契約に基づく乙の損害賠償責任の総額は、**損害の原因となった個別契約に定める報酬額（税抜）を上限**とする。
> 4. 前2項の規定は、**故意又は重大な過失による場合には適用しない**。

**交渉ポイント:**
- 通常損害に限定し、逸失利益等を除外
- 報酬額を上限として青天井を回避
- 重過失の場合は上限除外で甲の利益も保護

---

## 4. 契約期間条項の交渉パターン

### パターン4-1: 自動更新の削除 + 追加作業条項

**法務課の要望:**
自動更新条項（第2項）・個別契約存続条項（第3項）を削除

**乙の対案:**
```
・自動更新削除は受け入れ
・追加作業の有償対応条項を追加
```

**条文例:**
> 1. 本契約の有効期間は、本契約締結日から1年間とする。
> 2. **本契約の終了後に甲が成果物の修正、補修その他の追加作業を乙に依頼する場合は、別途個別契約を締結の上、有償にて対応するものとする。**

**交渉ポイント:**
- 自動更新がなくなっても、追加作業は有償で対応可能
- 「無料で直してくれ」を防止

---

### パターン4-2: 個別契約の存続規定

**法務課の要望:**
個別契約存続条項を削除

**乙の対案:**
```
・残存条項の中で個別契約の存続を規定
```

**条文例:**
> 2. 本契約の終了後も、**本契約の有効期間中に締結した個別契約については、当該個別契約の有効期間が終了するまでは本契約の規定が適用される**ものとする。

**交渉ポイント:**
- 進行中の個別契約が保護される
- 契約終了と同時に仕事が宙に浮くリスクを回避

---

## 5. 解除条項の交渉パターン

### パターン5-1: 無催告解除事由の具体化

**法務課の要望:**
「信頼関係を喪失する行為」を無催告解除事由に追加

**乙の対案:**
```
・抽象的な表現を具体化
```

**条文例:**
> ~~その他信頼関係を喪失する行為をしたとき~~
> **本契約又は個別契約の重要な条項に違反し、その違反が重大で是正が不可能であると合理的に判断されるとき**

**交渉ポイント:**
- 「信頼関係喪失」は主観的で争いの原因になりやすい
- 「重要な条項違反」「重大」「是正不可能」「合理的判断」で客観化

---

### パターン5-2: 任意解除の双方向化

**法務課の要望:**
甲（発注者）からの任意解除権（30日前通知）を新設

**乙の対案:**
```
・双方向の解除権に変更
・精算条項を追加
```

**条文例:**
> 3. **甲又は乙は**、相手方に対し、解除希望日の30日前までに書面（電磁的方法を含む。）による通知を行うことにより、本契約の全部又は一部を、理由のいかんを問わず解除することができるものとする。
> 4. **前項に基づき本契約が解除された場合、解除の効力発生日までに乙が提供した業務に対する報酬は、甲が速やかに支払うものとする。**

**交渉ポイント:**
- 甲のみに認めるのは不公平
- 「甲又は乙は」で双方向化
- 精算条項で「働いた分はもらえる」を確保

---

### パターン5-3: 個別契約の存続規定

**法務課の要望:**
任意解除時の個別契約存続規定を新設

**乙の対案:**
```
・そのまま受け入れ（乙に有利）
```

**条文例:**
> 5. 第3項に基づく本契約の解除は、解除の効力発生日前に成立した個別契約の効力には何ら影響を及ぼさないものとし、当該個別契約については、なお有効に存続するものとする。

**交渉ポイント:**
- 進行中の仕事が保護される
- 甲も乙もwin-win

## 交渉の基本姿勢

### DO（すべきこと）

| # | 内容 |
|---|------|
| 1 | 相手の要望の背景・意図を理解する |
| 2 | 「全面拒否」ではなく「条件付き受入」の姿勢で臨む |
| 3 | 受け入れられる項目を先に合意し、交渉力を温存 |
| 4 | 代替案・折衷案を準備して臨む |
| 5 | 双方にメリットがある形で提案する |

### DON'T（避けるべきこと）

| # | 内容 |
|---|------|
| 1 | 感情的な反応 |
| 2 | 法的根拠なき拒否 |
| 3 | 相手の法務担当者の顔をつぶす対応 |
| 4 | 一方的に有利な条項の押し付け |
| 5 | 重要項目と些末な項目を同列に扱う |

## メール回答のテンプレート

### 全体構成

```
○○様

いつも大変お世話になっております。[氏名]です。

この度は、取引基本契約書について、法務課の皆様に専門的な観点から
大変詳細かつ丁寧にご確認いただき、心より御礼申し上げます。

いただいたご指摘はいずれも的確なものであり、
御社の契約管理に対する真摯な姿勢に深く敬意を表します。

各項目について、以下のとおり回答させていただきます。


【①項目名】

＜ご確認事項＞
[法務課の要望を要約]

＜回答＞
[回答内容]


（以下、各項目を繰り返し）


以上、ご回答申し上げます。

一部、こちらからのお願いも含まれておりますが、
いずれも御社との良好な関係を長く続けていくために
必要な事項と考えております。

何卒ご検討いただけますようお願い申し上げます。

引き続きよろしくお願い申し上げます。

[氏名]
```

### 回答パターン別の表現

| パターン | 表現例 |
|---------|--------|
| そのまま受入 | 「ご指摘を承知いたしました。ご提案いただいた内容のとおり変更させていただきます。」 |
| 条件付き受入 | 「ご指摘を承知いたしました。恐れ入りますが、以下の内容を追加させていただければ幸いに存じます。」 |
| 理由説明 | 「御社のリスク管理のご趣旨は十分理解しておりますので、上記の形でご理解いただけますと幸いでございます。」 |
| 双方向化の提案 | 「双方に同じ権利があることで、公平性が担保されるものと存じます」 |
