# 取引基本契約書 変数テンプレート

> このテンプレートは`schemas/contract-input.json`の変数を使用して契約書を生成します。
> 変数形式: `{{変数パス}}` (例: `{{parties.orderer.name}}`)

---

## レイアウト安定化ガイド

DOCX/PDF変換時の崩れを防ぐため、以下のルールに従ってください。

### 文字数制限

| 変数 | 最大文字数 | 超過時の対処 |
|------|-----------|-------------|
| `parties.orderer.name` | 40 | 略称を使用または改行 |
| `parties.contractor.name` | 40 | 略称を使用または改行 |
| `parties.*.address` | 60 | 郵便番号を別行に |
| `business.services[].name` | 30 | 略称を使用 |
| `business.services[].description` | 100 | 複数文に分割 |

### 禁止文字

以下の文字は使用禁止（変換時に崩れる原因）：

- ①②③ → (1)(2)(3) に置換
- ㈱㈲ → 株式会社、有限会社 に置換
- Ⅰ Ⅱ Ⅲ → 第1、第2、第3 に置換
- 波ダッシュ「～」 → 全角チルダ「〜」に置換

### 改ページ制御

署名欄の前には必ず改ページを挿入（本テンプレート末尾に設定済み）：

```html
<div style="page-break-before: always; page-break-after: avoid;"></div>
```

### 検証コマンド

変数展開後、変換前に必ず検証を実行：

```bash
node scripts/validate-layout.mjs output/contract.md
```

## 第1章　総則

### 第1条（目的）

本契約は、甲が乙に対して発注する第2条に定める内容の業務（以下「本業務」という。）について、甲乙間の取引に関する基本的な事項を定めることを目的とする。

### 第2条（業務内容）

1. 乙が甲に対して提供する本業務の内容は、次のとおりとする。
{{#each business.services}}
   - ({{@index + 1}}) {{name}}
     - 業務内容: {{description}}
{{#if deliverables}}
     - 詳細・備考: {{deliverables}}
{{/if}}
{{/each}}
2. 本業務の詳細な仕様、納入条件その他の事項は、【別紙】取引条件明示書又は個別契約において定めるものとする。
3. 甲は、本業務の内容を変更する必要がある場合、乙と協議の上、書面により変更するものとする。

### 第3条（個別契約）

1. 本契約に定める事項は、甲乙間で別途締結される個別契約にも共通して適用される。なお、個別契約は、甲が乙に本業務を委託するにあたり、具体的な委託内容や取引条件を定め、乙がこれを受託した契約をいう。
2. 本業務の具体的な内容、報酬、納期等は、本契約に基づき、甲乙間で別途締結する個別契約において定める。
3. 個別契約は、書面、電子メール又はその他甲乙が合意した方法により締結することができる。
4. 個別契約の内容が本契約の内容と抵触する規定が定められている場合、当該抵触する部分については、個別契約が優先的に適用されるものとする。
5. 個別契約には、次の各号に掲げる事項を定めるものとする。
   - (1) 業務の内容及び範囲
   - (2) 成果物の仕様（該当する場合）
   - (3) 納入期日又は役務提供の期日
   - (4) 報酬の額及び支払期日
   - (5) その他甲乙が必要と認める事項

### 第4条（定義）

本契約において、次の各号に掲げる用語の意義は、当該各号に定めるところによる。

1. **「成果物」** 乙が本業務の遂行により作成するドキュメント、プログラム、データその他一切の成果をいう。
2. **「秘密情報」** 第18条に定義する情報をいう。
{{#if business.useAI}}
3. **「AI」** 人工知能（Artificial Intelligence）及び機械学習技術を利用したシステムをいう。
{{/if}}

## 第2章　業務の遂行

### 第5条（業務の遂行）

1. 乙は、善良なる管理者の注意をもって本業務を遂行する。
2. 乙は、本業務の遂行にあたり、甲の指示に従い、甲と緊密に連携するものとする。
3. 乙は、本業務の進捗状況について、甲の求めに応じて報告するものとする。

### 第6条（再委託）

1. 乙は、甲の事前の承諾を得て、乙と業務委託契約を締結した第三者（以下「再委託先」という。）に、本業務の遂行に必要な範囲で再委託することができる。
2. 乙は、再委託先に対して、自己が本契約において負う義務と同様の義務を負わせなければならず、かかる義務に再委託先が違反した場合、自己が違反したものとして甲に対して責任を負う。

{{#if business.useAI}}
### 第7条（AI利用に関する説明）

1. 乙は、本業務の遂行にあたりAIを利用する場合、甲に対し、その旨及びAIの利用範囲について説明するものとする。
2. 甲は、AI（{{business.aiServices.join("、")}}等の生成AIを含む。）の利用に関して、次の各号に掲げるリスクが存在することをあらかじめ認識し、これを承諾した上で乙に本業務を委託するものとする。
   - (1) **技術的制約によるリスク**: 生成AIの出力には事実と異なる内容が含まれる可能性があること（いわゆるハルシネーション）、及び出力結果には技術的な限界があること
   - (2) **セキュリティリスク**: 生成AIサービスの利用に伴う情報漏洩、サイバー攻撃、不正アクセス等のセキュリティ上のリスクが存在すること
   - (3) **知的財産権侵害リスク**: 生成AIの出力が、第三者の著作権、商標権、特許権その他の知的財産権を侵害する可能性があること
   - (4) **法的リスク**: AI技術に関する法規制が発展途上にあり、将来の法改正等により利用条件が変更される可能性があること
3. 甲は、成果物の利用にあたっては、前項各号に掲げるリスクを踏まえ、自己の責任において内容を確認し、必要に応じて専門家への相談その他の措置を講じるものとする。
4. 乙は、AIを利用する場合であっても、本業務の成果について善管注意義務の範囲内で責任を負う。ただし、次の各号に掲げる場合に生じた損害については、乙は責任を負わない。
   - (1) 甲がAIの利用を指示又は承諾した場合
   - (2) 甲が成果物の内容を確認せずに利用したことに起因する場合
   - (3) 第2項各号に掲げるリスクに起因する損害であって、乙が善管注意義務を尽くしていた場合
{{#if confidentiality.aiLearningProhibited}}
5. 乙は、甲の秘密情報を生成AIの学習データとして使用しないものとする。
{{/if}}

## 第7章　秘密保持

### 第{{business.useAI ? 18 : 17}}条（秘密情報）

1. 本契約において「秘密情報」とは、本契約又は個別契約に関連して、一方当事者が相手方に対し、書面、電子データその他の媒体により開示した技術上又は営業上の情報であって、開示の際に秘密である旨を明示したものをいう。
2. 前項の規定にかかわらず、次の各号のいずれかに該当する情報は秘密情報に含まれない。
   - (1) 開示を受けた時点で既に公知であった情報
   - (2) 開示を受けた後、受領者の責めによらず公知となった情報
   - (3) 開示を受けた時点で既に受領者が保有していた情報
   - (4) 受領者が第三者から秘密保持義務を負うことなく正当に取得した情報
   - (5) 受領者が独自に開発した情報

### 第{{business.useAI ? 19 : 18}}条（秘密保持義務）

1. 甲及び乙は、相手方の秘密情報を厳に秘密として保持し、相手方の事前の書面による承諾なく、第三者に開示又は漏洩してはならない。
2. 甲及び乙は、秘密情報を本契約又は個別契約の目的以外に使用してはならない。
3. 甲及び乙は、法令に基づき又は裁判所若しくは行政機関から秘密情報の開示を命じられた場合、必要最小限の範囲で開示することができる。この場合、可能な限り事前に相手方に通知するものとする。

4. 甲及び乙は、相手方の秘密情報を、生成AI等の機械学習の学習データとして使用してはならない。
{{/if}}
