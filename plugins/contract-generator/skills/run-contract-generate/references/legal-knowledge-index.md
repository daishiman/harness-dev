# 法令ナレッジ索引(Progressive Disclosure)

本スキルは法務承認済 `.docx` ひな形を**改変しない**ため、契約条文の法的判断は最小限。
ただし差込内容のレビューや、ひな形改定時の妥当性確認のため、参考スキルの法令ナレッジを索引化しておく。
必要時のみ参照する(Progressive Disclosure)。

正本: `doc/参考Skill/contract-generator/references/`(対話駆動の旧スキル。条文知識のみ流用)

## 本スキルでの適用方針(参考スキルから継承)

- 法令条文は番号を明記して引用する(著作権法27/28条・フリーランス法第4条等)。
- 不確実事項は限定詞(「可能性があります」)を使う。法的有効性は保証しない。
- 最新法令は必要時 WebSearch で確認(2026以降のフリーランス新法追加省令など)。
- 個人委託(個人②)はフリーランス新法の明示義務が条文に内蔵済み。法人委託(法人①)は対象外。
