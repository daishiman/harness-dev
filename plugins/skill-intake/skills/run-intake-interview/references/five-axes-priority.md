# Five Axes Priority (interview)

5 軸の処理順序は固定。

| 順位 | 軸 | 質問例 (beginner tier) |
|--|--|--|
| 1 | 出力先 | できあがった結果はどこに置きたいですか? |
| 2 | 情報源 | 元のデータは何から取りますか? |
| 3 | 共有相手 | 結果を見せる相手は誰ですか? |
| 4 | 真の課題 | 今、一番困っていることは何ですか? |
| 5 | ナレッジ資産 | 過去にまとめた資料・ノウハウはありますか? |
