# 外部連携カタログ

`skill-intake-option-presenter` SubAgent が提示する選択肢の正本。各サービスで「ここまで取れる／ここまで自動化できる」を平易に書く。

## サービス一覧

| サービス | 一言 | 主用途 | 認証 |
|----------|------|--------|------|
| Google Forms | 申込フォーム | 質問・回答収集 | OAuth |
| Google Drive | クラウド保管 | ファイル保存・共有 | OAuth |
| Google Sheets | 表計算 | 集計・一覧化 | OAuth |
| Google Calendar | 予定表 | スケジュール取得・登録 | OAuth |
| Gmail | メール | 送受信・検索 | OAuth |
| Slack | チャット | 通知・対話 | Bot Token |
| Notion | ノート / DB | ページ・データベース | PAT or Internal Integration |
| Discord | コミュニティ | スレッド配信 | Webhook / Bot |
| GitHub | コード / Issue | リポジトリ操作 | PAT |
| OpenAI | AI 生成 | 要約・生成・Whisper | API Key |
| Anthropic | AI 生成 | 要約・生成 | API Key |
| arxiv | 論文 DB | 論文検索・取得 | 無認証 |
| Whisper | 音声→文字 | 文字起こし | API Key（OpenAI） |

---

## Google Forms

| 観点 | 内容 |
|------|------|
| ここまで取れる | フォームの構造・回答一覧・回答件数 |
| ここまで自動化できる | フォームの新規作成・質問追加・回答先 Sheets 指定 |
| 制限 | 一部の高度 UI（分岐質問の細部）は API 非対応 |
| 平易な説明 | Google が用意してくれる「申込書つくる箱」 |

## Google Drive

| 観点 | 内容 |
|------|------|
| ここまで取れる | フォルダ一覧・ファイルメタ情報・本文 |
| ここまで自動化できる | フォルダ作成・ファイル配置・共有設定 |
| 平易な説明 | みんなで使うクラウドの本棚 |

## Google Sheets

| 観点 | 内容 |
|------|------|
| ここまで取れる | シート全体・特定セル・関数結果 |
| ここまで自動化できる | 値の書込・行追加・条件付き書式 |
| 平易な説明 | Excel のオンライン版。自動で行を増やしたり集計したり |

## Google Calendar

| 観点 | 内容 |
|------|------|
| ここまで取れる | 予定一覧・参加者・繰返ルール |
| ここまで自動化できる | 予定作成・更新・招待送信 |
| 平易な説明 | カレンダーに自動で予定を入れたり、来週の予定を取り出したり |

## Gmail

| 観点 | 内容 |
|------|------|
| ここまで取れる | メール本文・添付・スレッド |
| ここまで自動化できる | 送信・下書き作成・ラベル付け |
| 制限 | 認証スコープに敏感 |
| 平易な説明 | メールを自動で送ったり、特定の差出人だけ抜き出したり |

## Slack

| 観点 | 内容 |
|------|------|

| ここまで自動化できる | メッセージ送信・予約投稿・リアクション |
| 平易な説明 | チームのチャットに自動通知 |

## Notion

| 観点 | 内容 |

|------|------|

## 提示テンプレート（option-presenter 用）

```
あなたの「<目的>」を満たすには、以下の選択肢があります。

1. <サービス A> — <平易な説明>
   メリット: <一言>
   制限: <一言>

2. <サービス B> — ...

おすすめ: <理由付きで 1 つ選ぶ>
気になるものはありますか？「わからない」で OK です。
```

## 制約フィルタ

`user_profile.constraints` に応じて自動除外。

| 制約 | 除外対象例 |
|------|------------|
| 予算ゼロ | OpenAI/Anthropic/Whisper（無料枠超過時） |
| 権限なし | 会社管理の Google アカウント連携 |
| 知識なし（コード書けない） | 自前 GitHub Actions |

## 組合せパターン（よく出る）

| パターン | 構成 | 例スキル |
|----------|------|----------|
| 申込集計 | Forms + Sheets + Drive | google-forms-generator |
| 自動収集配信 | API + LLM + Discord | arxiv-paper-reporter |
| 議事録 | Drive + Whisper + LLM + Discord | google-meet-minutes-generator |
| 報告書 | Sheets + LLM + Notion | （新規候補） |
